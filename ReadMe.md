# Программа управления памятью

**Сборка из директории ThirdLab:** ./clear_build.sh

**Описание:**
В программе представленый собственный аллокатор, стратегии и контейнер для работы с памятью.
Содержатся две стратегии пула и линейная.
Стратегия Pool отлично подходит для Node-based контейнеров типа list. map и тд. и не подходит для линейных контейнеров.
Поэтому создана линейная стратегия, которая позволяет перегрузить оператор индексации.
Собственный контейнер - вектор-подобный.
Аллокатор представлен как менеджер, способный принять разные стратегии.
Так же реализованы простейшие юнит тесты для контроля нормального функционирования.
